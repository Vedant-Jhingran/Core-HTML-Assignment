1. p tag <p></p>: The <p> tag defines a paragraph. Browsers automatically add a single blank line before and after each <p> element.

  2. b tag <b></b>: The <b> tag in HTML is used to specify the bold text without any extra importance. The text is written within <b> tag display in bold size.

  3. i tag <i></i>: The <i> tag in HTML is used to italicize the text without any extra importance. The text is written within <i> tag display as italicized.
